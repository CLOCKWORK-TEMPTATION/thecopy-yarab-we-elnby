# ADR-004: Tailwind + Radix + shell مخصص للمحرر

## التاريخ
2026-03-30

## الحالة
مقبول

## السياق
المشروع يحتاج واجهات عامة متعددة، ومحرراً عربياً بطابع بصري مخصص.

## القرار
استخدام Tailwind وRadix للمكونات العامة، مع shell مخصص في editor.

## البدائل المدروسة

| البديل | المزايا | العيوب |
|---|---|---|
| Tailwind + Radix | مرونة عالية واتساق جيد | مسؤولية تركيب أعلى |
| مكتبة UI كاملة مثل MUI | سرعة أكبر لبعض الشاشات | لا تطابق الأسلوب الحالي للمحرر |

## النتائج
الواجهة المرئية متسقة مع احتياجات editor، لكن المكونات تتوزع بين shared/ui وeditor shell بشكل يحتاج توثيقاً جيداً.
