# ADR-001: اعتماد App Router في الويب

## التاريخ
2026-03-30

## الحالة
مقبول

## السياق
تطبيق الويب مبني بالكامل حول `apps/web/src/app` مع layouts متداخلة ومسارات مجمعة مثل `(main)`.

## القرار
اعتماد App Router كنمط التوجيه الرسمي.

## البدائل المدروسة

| البديل | المزايا | العيوب |
|---|---|---|
| App Router | layouts متداخلة، route handlers، تنظيم أوضح للتطبيقات | يتطلب انضباطاً أعلى في Server/Client boundaries |
| Pages Router | أبسط تاريخياً | لا يتوافق مع البنية الحالية للمجلدات |

## النتائج
انسجام كامل مع البنية الحالية، لكن يزيد حساسية المكونات التي تعتمد DOM مثل editor ويجعل بعضها client-only.
