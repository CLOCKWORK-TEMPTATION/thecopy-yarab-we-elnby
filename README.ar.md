# النسخة

**ورشة سينمائية عربية متكاملة من السطر الأول للميزانية.**

تكتب السيناريو، تُفكّكه، تُسعّره، وتُخرجه بصرياً — كله بالعربية، كله في شاشة واحدة.

---

## لماذا وُجدت النسخة

أنت تكتب دراما عربية. أدواتك ليست كذلك.

السيناريو في
```text
Word
```
لأن
```text
Final Draft
```
لا يفهم الترقيم العربي. الميزانية في
```text
Excel
```
لأن
```text
StudioBinder
```
بالدولار. التفكيك على
```text
PDF
```
ممسوح ضوئياً. اللوحة البصرية على
```text
Pinterest
```
بمراجع غربية. مساعد الذكاء الاصطناعي يقترح حواراً يشبه مسلسلاً أمريكياً مدبلجاً.

كل أداة بلغة، كل أداة بحساب، كل أداة بسعر، ولا أداة منهم وُلدت لتفهم أن «الفجر» ليس
```text
dawn
```
وأن «المصرية» ليست لهجة بل عالم.

النسخة هي الأداة الواحدة التي تتكلم لغتك من البداية.

## ابدأ في ثلاث خطوات

```bash
# 1. استنسخ المستودع وثبّت التبعيات
git clone https://github.com/CLOCKWORK-TEMPTATION/yarab-we-elnby-thecopy.git
cd yarab-we-elnby-thecopy
pnpm install

# 2. انسخ ملف البيئة وأضف مفاتيح الذكاء الاصطناعي
cp .env.example .env
# عدّل GEMINI_API_KEY و TIPTAP_PRO_TOKEN في .env

# 3. شغّل المنصة كاملة
pnpm dev
```

افتح
```text
http://localhost:5000
```
وستجد ورشتك جاهزة.

## ماذا تفعل بها

### تكتب بطلاقة، يساعدك ذكاء يفهم الدراما العربية

محرر نصوص يفتح بسرعة، يحفظ تلقائياً، ويقترح عليك تطوير الحوار والشخصية بنماذج
```text
Gemini
```
و
```text
Claude
```
و
```text
GPT
```
في نفس الواجهة. تختار النموذج الذي يناسب اللحظة، تستبدله متى شئت، وتظل تكتب بلا انقطاع.

### تُفكّك السيناريو بضغطة، لا بأسبوع

تستخرج الشخصيات، المواقع، الإكسسوار، والملابس من نصك تلقائياً. ما كان يأخذ من مساعد الإخراج خمسة أيام يصير جاهزاً قبل القهوة الثانية.

### تُسعّر الإنتاج قبل أن يبدأ

أداة الميزانية تربط كل لقطة بتكلفتها، وتُخرج لك تقديراً يفهمه المنتج العربي بعملته المحلية وأسعار سوقه — لا بـ
```text
Hollywood rates
```
لا تشبه واقعنا.

### تبني عالم الفيلم بصرياً قبل الكاميرا

استوديو المخرجين يُولّد لوحات بصرية، يقترح زوايا تصوير، ويبني
```text
storyboard
```
يفهم أن المشهد يدور في حي شعبي لا في
```text
suburban Los Angeles
```

## الأدوات داخل المنصة

| الأداة | ماذا تحل |
|---|---|
| محرر السيناريو | كتابة بصيغة احترافية مع مساعد ذكي يفهم الحوار العربي |
| تفكيك السيناريو | استخراج تلقائي للعناصر الإنتاجية من النص |
| الميزانية | تسعير دقيق مرتبط بكل لقطة وكل عنصر |
| استوديو المخرجين | لوحات بصرية وتخطيط تصوير |
| العصف الذهني | تطوير فكرة من سطر واحد إلى معالجة كاملة |
| المدقق الأسلوبي | مراجعة الإيقاع، الاتساق، وعمق الشخصية |
| ذاكرة المشروع | كل ما كتبته يبقى متاحاً للذكاء الاصطناعي ليفهم سياقك الكامل |

## ما تتجنّبه باستخدامها

- التنقّل بين خمسة برامج لإنجاز ملف واحد.
- الترجمة الذهنية المستمرة من العربية إلى الإنجليزية لتفهمك الأداة.
- اشتراكات شهرية بالدولار لأدوات صُمّمت لسوق غير سوقك.
- ذكاء اصطناعي يقترح حواراً «ترجمياً» لا يُقال على لسان شخصية مصرية أو شامية أو خليجية.
