# مرآة Claude للمستودع

هذا الملف ليس العقد الأعلى للمستودع.

ابدأ دائمًا من:

```text
AGENTS.md
```

ثم اقرأ:

```text
output/session-state.md
```

ثم ارجع إلى:

```text
.repo-agent/OPERATING-CONTRACT.md
.repo-agent/STARTUP-PROTOCOL.md
.repo-agent/HANDOFF-PROTOCOL.md
```

إذا بدأت الجلسة من أداة تقرأ هذا الملف مباشرة:

1. اقرأ العقد الأعلى.
2. اقرأ الحالة الحية.
3. أخرج brief قصيرًا من 3 إلى 7 حقائق تشغيلية.
4. ثم فقط ابدأ العمل.

ممنوع اعتبار هذا الملف مصدر حقيقة مستقلًا أو مكانًا للحالة التشغيلية.

وعند نهاية الجولة:

1. حدّث:

```text
output/round-notes.md
```

2. حدّث:

```text
output/session-state.md
```

إذا تغيّرت الحقيقة التشغيلية أو البنيوية.
3. أخرج handoff brief قصيرًا يذكر ما تغيّر وما ثبت وما بقي مفتوحًا.
