## 3) المشكلات القابلة لإعادة الإنتاج — مع الخطوات

**خطأ تصدير PDF** (UX-12 / DATA-03):
1. افتح `https://www.thecopy.app/editor`
2. اكتب أي محتوى أو استخدم المحتوى الموجود
3. ملف ← تصدير PDF
4. **النتيجة:** Toast أحمر "تعذّر التصدير — Attempting to parse an unsupported color function 'oklch'"
5. **السبب التقني:** الموقع يستخدم `oklch()` في CSS variables، ومكتبة تصدير PDF (jsPDF/html2canvas أو ما شابه) لا تدعم هذا التنسيق

**عداد الكلمات لا يتحدّث** (UX-08):
1. افتح المحرر
2. سجّل عدد الكلمات الحالي في شريط الحالة السفلي
3. اكتب عدة كلمات متباعدة بمسافات
4. **النتيجة:** عدّاد الحروف يتحدّث، عدّاد الكلمات يظل ثابتاً

**فقدان أقسام الشريط الجانبي بعد البحث** (UX-11):
1. في الشريط الجانبي، اكتب أي نص في حقل البحث (مثل "مشروع اختبار")
2. تختفي أقسام: المستندات الأخيرة، المشاريع، المكتبة
3. امسح حقل البحث يدوياً (triple-click + Delete)
4. **النتيجة:** الأقسام لا تعود؛ يبقى فقط "الإعدادات" + رسالة "لا توجد نتائج"
5. الحل المؤقت: إعادة تحميل الصفحة

**رفض الملفات غير المدعومة** (VAL-07 — هذا اختبار ناجح):
1. ملف ← إدراج ملف...
2. ارفع ملف بامتداد `.exe`
3. **النتيجة الصحيحة:** Toast واضح يرفض الملف ويذكر التنسيقات المدعومة (PDF, DOC, DOCX, TXT, Fountain, FDX)
