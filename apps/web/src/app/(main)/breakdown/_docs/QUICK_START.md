# 🚀 بدء سريع - BreakBreak AI

**وقت القراءة**: 5 دقائق  
**المتطلبات**: Node.js ≥ 16

---

## 1️⃣ إعداد الملفات

### على Windows (PowerShell)

```powershell
# تحميل المتعلقات
npm install

# نسخ قالب المتغيرات
Copy-Item .env.example .env.local

# فتح الملف للتعديل
notepad .env.local
```

### على Mac/Linux (Bash)

```bash
# تحميل المتعلقات
npm install

# نسخ قالب المتغيرات
cp .env.example .env.local

# فتح الملف للتعديل
nano .env.local
```

## 2️⃣ تعيين API Key

### احصل على API Key مجاني من Google

1. اذهب إلى: https://aistudio.google.com/apikey
2. انقر "Create API Key"
3. سيظهر المفتاح - **انسخه**

### في ملف `.env.local`

```dotenv
# قبل:
GEMINI_API_KEY=PLACEHOLDER_API_KEY

# بعد:
GEMINI_API_KEY=AIzaSyDx... (الرمز الفعلي الخاص بك)
```

⚠️ **تحذير أمني**: لا تشارك هذا المفتاح مع أحد!

## 🐛 استكشاف الأخطاء

### ❌ "GEMINI_API_KEY is required"

**السبب**: المفتاح غير معيّن

**الحل**:

1. تحقق من وجود `.env.local`
2. تأكد أن المفتاح ليس `PLACEHOLDER_API_KEY`
3. أعد تشغيل الخادم (`Ctrl+C` ثم `npm run dev`)

---

### ❌ "Failed to segment script"

**السبب**: قد يكون الملف الحد ثقيل أو الاتصال ضعيف

**الحل**:

1. جرب سيناريو أقصر (3-5 مشاهد)
2. تأكد من الاتصال بالإنترنت
3. جرب من جديد

---

### ❌ "ChatBot not responding"

**السبب**: مشكلة في الاتصال بـ API

**الحل**:

1. افتح Developer Tools (`F12`)
2. اذهب إلى `Console` tab
3. ابحث عن الرسائل الحمراء (errors)
4. أعد تحميل الصفحة (`F5`)
