# PRD مختصر تنفيذي

## الرؤية

بناء منصة ويب تشغيلية لمساعد مدير التصوير، تربط النص، التخطيط البصري، يوم التصوير، الرشز، الاستمرارية، والتسليم إلى ما بعد الإنتاج.

## المستخدمون

- مدير التصوير
- المخرج
- DIT
- الغافر
- مشرف الاستمرارية
- المونتير
- الملوّن
- مشرف VFX
- المنتج
- مسؤول النظام

## المجالات الأساسية

1. إدارة المشاريع والهوية والصلاحيات
2. الأصول والمعرفة والبحث
3. النص والمشاهد والنسخ
4. الخطط البصرية وخطط الكاميرا والإضاءة
5. التغطية والـsetups والـshots
6. يوم التصوير والصوت والمزامنة
7. الـtakes وmetadata وDIT
8. الاستمرارية والرشز والاستكمال
9. handoff إلى editorial / color / VFX
10. تشغيل الوكلاء والتقييمات والحوكمة

## مبادئ المنتج

- العربية أصلية وليست مترجمة شكلياً
- كل مخرجات الذكاء الاصطناعي مسودات منظمة قابلة للمراجعة
- لا تعديل مباشر على كيان معتمد
- البحث يدعم العربية والإنجليزية والهجين المهني
- العمل الميداني يحتاج Offline-first ومزامنة لاحقة
