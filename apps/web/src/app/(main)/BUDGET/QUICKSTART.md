# 🚀 البدء السريع - FilmBudget AI Pro

## 📝 خطوات البدء في 3 دقائق

### 1️⃣ التحضير (دقيقة واحدة)

```bash
# افتح PowerShell واذهب للمجلد
cd "D:\New folder (58)\the...copy\frontend\src\app\(main)\BUDGET"

# تأكد من تثبيت المكتبات (إذا لم يتم بعد)
npm install
```

---

### 2️⃣ إعداد API Key (30 ثانية)

1. احصل على مفتاح مجاني من: https://makersuite.google.com/app/apikey
2. افتح ملف `.env.local`
3. أضف المفتاح:

```env
NEXT_PUBLIC_GEMINI_API_KEY=AIza...your_key_here
```

---

### 3️⃣ التشغيل (30 ثانية)

```bash
npm run dev
```

✅ افتح المتصفح: **http://localhost:3001**

---

1. **عنوان الفيلم:**

   ```
   رحلة في الصحراء
   ```

   ```
   فيلم وثائقي مدته 90 دقيقة

   المواقع:
   - صحراء سيناء (5 أيام تصوير)
   - واحة سيوة (3 أيام تصوير)

   الطاقم:
   - مخرج + مساعد مخرج
   - مصور رئيسي + مصور ثاني
   - مهندس صوت + مساعد صوت
   - منتج + منسق إنتاج

   المعدات:
   - كاميرا RED Dragon 6K
   - درون للتصوير الجوي
   - معدات صوت احترافية

   الممثلون:
   - راوي رئيسي
   - 5 شخصيات محلية

   الميزانية المتوقعة: 80,000 - 100,000 دولار
   ```

3. **انقر:** "إنشاء الميزانية الكاملة"

4. **انتظر 30 ثانية** وستحصل على ميزانية تفصيلية!

## 🔧 حل المشاكل السريع

### المشكلة: "Gemini API Key is missing"

**الحل:** أضف المفتاح في `.env.local`

### المشكلة: Port 3000 مستخدم

**الحل:** التطبيق سيستخدم 3001 تلقائياً ✅

### المشكلة: خطأ في البناء

**الحل:**

```bash
rm -rf .next
npm run build
```
